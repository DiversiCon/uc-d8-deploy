# Details component

Displays details about a campus house. 

## Usage

```html
<div class="c-details">
  <div class="c-details__content">
    <ul class="c-details__stats">
      <li class="c-details__stats-item"><span class="c-details__number">9</span> <span class="c-details__label">Houses</span></li>
      <li class="c-details__stats-item"><span class="c-details__number">62</span> <span class="c-details__label">Rooms</span></li>
    </ul>
    <ul class="c-details__info">
      <li class="c-details__info-item"><span class="c-details__label">Room Types:</span> <span class="c-details__description">22 Singles, 40 Doubles</span></li>
      <li class="c-details__info-item"><span class="c-details__label">Dining Hall:</span> <span class="c-details__description">Arley D. Cathey Residential Dining</span></li>
      <li class="c-details__info-item"><span class="c-details__label">Built:</span> <span class="c-details__description">2016</span></li>
    </ul>
  </div>
</div>
```
