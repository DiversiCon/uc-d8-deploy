# Image List component

This component allows a list of images to be placed on a page. The heading and subheading are both optional. The images can be links.

## Usage

```html
<div class="c-image-list">
  <h2 class="c-image-list__heading">House Communities</h2>
  <h3 class="c-image-list__subheading">East</h3>
  <ul class="c-image-list__list">
    <li class="c-image-list__item">
      <a href="#" class="c-image-list__link">
        <img src="/themes/custom/uccollege/it_showcase/images/crests/yuen.png" alt="YUEN" title="YUEN" class="c-image-list__img">
      </a>
    </li>
    <li class="c-image-list__item">
      <img src="/themes/custom/uccollege/it_showcase/images/crests/boyer.png" alt="BOYER" title="BOYER" class="c-image-list__img">
    </li>
    <li class="c-image-list__item">
      <a href="#" class="c-image-list__link">
        <img src="/themes/custom/uccollege/it_showcase/images/crests/behar.png" alt="BEHAR" title="BEHAR" class="c-image-list__img">
      </a>
    </li>
    <li class="c-image-list__item">
      <img src="/themes/custom/uccollege/it_showcase/images/crests/dougan.png" alt="DOUGAN-NIKLASON" title="DOUGAN-NIKLASON" class="c-image-list__img">
    </li>
  </ul>
</div>
```
